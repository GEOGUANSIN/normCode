# 持有建议生成提示

根据中性/混合信号生成持有建议。

## 输入
**验证信号包**: $input_1

## 任务
1. 解释为何信号不支持方向性交易
2. 识别需要什么变化才能产生买入或卖出信号
3. 建议监控行动

## 输出格式
返回一个包含"answer"字段的JSON对象，该字段包含建议：
```json
{
  "answer": {
    "action": "HOLD",
    "rationale": "<对中性信号解读的说明>",
    "confidence": <0.0到1.0之间的浮点数>,
    "wait_for": [<会触发行动的条件列表>],
    "monitoring_points": [<需关注的事项>]
  }
}
```
