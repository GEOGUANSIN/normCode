# 卖出建议生成提示

根据检测到的看跌信号生成卖出建议。

## 输入
**验证信号包**: $input_1

**投资者风险偏好**: $input_2

## 任务
1. 将看跌信号综合成连贯的退出/做空理由
2. 考虑投资者的风险约束
3. 仓位管理将由仓位计算脚本单独计算

## 输出格式
返回一个包含"answer"字段的JSON对象，该字段包含建议：
```json
{
  "answer": {
    "action": "SELL",
    "rationale": "<详细解释为何信号支持卖出>",
    "confidence": <0.0到1.0之间的浮点数>,
    "key_drivers": [<主要看跌驱动因素列表>],
    "risk_factors": [<不卖出的风险列表>],
    "suggested_exit": "<退出策略描述>"
  }
}
```
