# 买入建议生成提示

根据检测到的看涨信号生成买入建议。

## 输入
**验证信号包**: $input_1

**投资者风险偏好**: $input_2

## 任务
1. 将看涨信号综合成连贯的投资理由
2. 考虑投资者的风险约束
3. 仓位管理将由仓位计算脚本单独计算

## 输出格式
返回一个包含"answer"字段的JSON对象，该字段包含建议：
```json
{
  "answer": {
    "action": "BUY",
    "rationale": "<详细解释为何信号支持买入>",
    "confidence": <0.0到1.0之间的浮点数>,
    "key_drivers": [<主要看涨驱动因素列表>],
    "risk_factors": [<需监控的风险列表>],
    "suggested_entry": "<入场策略描述>"
  }
}
```
