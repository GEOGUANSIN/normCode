# 最终投资决策综合提示

将所有建议综合成最终投资决策。

## 输入
**所有建议**: $input_1 

**投资者风险偏好**: $input_2

## 任务
1. 审查所有生成的建议（看涨/看跌/中性 - 只有一个应该是活跃的）
2. 将投资者的约束作为最终过滤条件应用
3. 生成权威的投资决策

## 需应用的约束条件
- 遵守最大持仓规模限制
- 应用止损规则
- 考虑投资期限匹配度
- 检查杠杆和做空许可

## 输出格式
返回一个包含"answer"字段的JSON对象，该字段包含决策：
```json
{
  "answer": {
    "final_action": "<BUY|SELL|HOLD>",
    "position_size_usd": <遵守限制的计算后仓位规模>,
    "stop_loss_price": <计算后的止损价位>,
    "target_price": <如适用的目标价>,
    "rationale": "<综合决策理由>",
    "confidence": <0.0到1.0之间的浮点数>,
    "constraints_applied": [<影响决策的约束条件列表>],
    "execution_notes": "<任何执行指导>"
  }
}
```
