# 情绪提取提示

你是一位专注于黄金市场的金融分析师。分析以下新闻文章并提取情绪信号。

## 输入
**新闻数据**: $input_1

## 任务
1. 对于每篇文章，评估其对黄金价格的情绪（看涨/看跌/中性）
2. 识别关键主题（货币政策、汇率动态、地缘政治风险等）
3. 汇总形成整体叙事信号

## 评估标准
考虑以下标准黄金市场驱动因素：
- **货币政策**：美联储利率决策、鸽派/鹰派信号
- **汇率动态**：美元强/弱势
- **通胀预期**：CPI数据、通胀对冲需求
- **地缘政治风险**：避险需求触发因素
- **央行活动**：官方部门买卖行为

## 输出格式
返回一个包含"answer"字段的JSON对象，该字段包含分析结果：
```json
{
  "answer": {
    "overall_score": <-1.0（非常看跌）到1.0（非常看涨）之间的浮点数>,
    "confidence": <0.0到1.0之间的浮点数>,
    "themes": [<已识别主题列表>],
    "summary": "<叙事信号的一段式总结>",
    "article_scores": [
      {"headline": "...", "score": <浮点数>, "key_driver": "..."}
    ]
  }
}
```
