How It Works

NormCode makes data flow explicit and isolated. Here's a simple example:

<- executive summary
    <= generate summary from flagged items
    <- discrepancy flags
        <= check for mismatches
        <- extracted figures
            <= extract financial data
            <- raw document
        <- database results

Reading bottom-up: The extraction step sees only the raw document. The mismatch check sees only extracted figures + database results. The summary step sees only discrepancy flags.
